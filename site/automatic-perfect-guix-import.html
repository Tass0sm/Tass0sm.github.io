Automatic, Perfect Guix Import

  • Conceived - June 12, 2021
  • Purpose: Replace every package manager with GNU Guix.

Summary

I'd love to replace every domain-specific package manager with GNU Guix. Is it possible to automatically take any package from one of these domain-specific package registries and convert it to a Guix package? Then one could have a Guix channel that lazily mirrors another repository like PyPi or Rubygems. That would be fantastic.
