I made a website with Haunt

I made this website because I'm considering taking on more development projects and thought this would be a good place to talk about them. I don't plan on using this website for much else but I might occasionally discuss some standalone topics that I find interesting.

I'm thinking of doing a "12 projects in 12 months" thing like Yahtzee Croshaw. He did games, but I think I will just focus on normal programs for general use or development. I'll probably use this website to document that. I also might document my experiences with other software and lessons from Computer Science.

Right now I'm looking to try the Guix package manager. I tried Nix before, but I had to abandon it due to the learning curve and lack of quality documentation. From what I hear Guix might be worse in that respect, but at least it is using Lisp. Nix is written in C++ and uses boost, which I really dislike.
